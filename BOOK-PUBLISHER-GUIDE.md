# Nadvi Book Publisher

## Publish a new book

1. Put the finished cover image in the `Books to Upload` folder. JPG, PNG, and WebP files are accepted; keep each image below 8 MB.
2. Double-click `Start Book Publisher.bat`.
3. Your private publishing form opens in the browser.
4. Choose the cover, complete the book details, and click **Publish to Website**.
5. Keep the black launcher window open until the green success message appears.
6. Wait about 1-3 minutes, then check `www.nadvipublications.online`.

The publisher adds the cover and details to GitHub. Vercel then updates the website automatically. Close the black launcher window after publishing.

## Important

- This tool is available only on this computer; it is not a public admin page.
- Upload public cover images only. Keep full PDF manuscripts in a private folder.
- Do not rename or move the website folder, launcher, `data` folder, or `public/books` folder.
- Publish one book at a time.
- If a red error appears, do not repeatedly click Publish. Take a screenshot of the message so it can be corrected safely.
